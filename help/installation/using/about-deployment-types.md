# 关于部署类型{#about-deployment-types}



Adobe Campaign的模块化设计支持范围广泛的部署配置，从独立设置（一台计算机上所有组件）到使用多个服务器的完全冗余和分布式架构的企业部署。 所有这些都取决于所需的性能和安全级别。

如果配置在多台计算机上，则在整个过程中不必使用相同的操作系统：例如，您可以在Linux + Apache上使用重定向服务器，在Windows上使用投放服务器。

>
