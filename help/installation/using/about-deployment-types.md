# 关于部署类型{#about-deployment-types}

Adobe Campaign的模块化设计支持各种部署配置，从独立设置（一台计算机上的所有组件）到使用多台服务器实现完全冗余和分布式架构的企业部署。 这一切取决于所需的性能和安全级别。

如果在多台计算机上进行配置，则不必在以下过程中使用相同的操作系统：例如，您可以在Linux + Apache上使用重定向服务器和Windows上的交付服务器。
