# Campaign Classic 选项列表{#configuring-campaign-options}

**[!UICONTROL Administration / Platform / Options]**&#x200B;节点允许您配置Adobe Campaign选项。

>[!NOTE]
>
>修改或更新Adobe Campaign选项只能由专家用户执行。

其中一些是内置的，安装活动时，还有一些可以根据需要手动添加。 可用选项因随实例安装的包而异。
