# 方法元素 {#methods--element}

![](../../../assets/v7-only.svg)

## 内容模型 {#content-model-11}

方法：==方法

## 属性 {#attributes-11}

无

## 父项 {#parents-11}

`<srcschema>`

## 子项 {#children-11}

方法

## 说明 {#description-11}

利用此元素，可定义 `<method>`  元素。 声明方法时必须使用。

## 属性说明 {#attribute-description-11}

此元素没有属性。

## 示例 {#examples-8}

```
<methods async="true"
...// definition of one or more <method
</methods>
```
