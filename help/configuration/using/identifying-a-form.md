# 识别表单{#identifying-a-form}

![](../../assets/common.svg)

通过输入表单，您可以从Adobe Campaign客户端控制台编辑与数据架构关联的实例。

表单通过其名称和命名空间进行标识。

表单的标识键是由命名空间和名称之间用冒号(“：”)分隔的字符串组成(例如：“cus:contact”)。
