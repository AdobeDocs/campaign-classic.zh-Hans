# 识别表单{#identifying-a-form}



输入表单允许您从Adobe Campaign客户端控制台编辑与数据架构关联的实例。

表单由其名称和命名空间进行标识。

表单的标识键是由命名空间和名称组成的字符串，以冒号(“：”)分隔（例如：“cus：contact”）。
