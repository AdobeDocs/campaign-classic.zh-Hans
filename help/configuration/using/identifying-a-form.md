# 识别窗体{#identifying-a-form}

输入表单允许您从Adobe Campaign客户端控制台编辑与模式关联的实例。

表单由其名称和命名空间标识。

表单的标识键是由命名空间和名称以冒号(“:”)分隔的字符串组成(例如：“cus:contact”)。
