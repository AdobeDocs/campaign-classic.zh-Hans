# 识别表单{#identifying-a-form}

通过输入表单，您可以从Adobe Campaign客户端控制台编辑与数据架构关联的实例。

表单由其名称和命名空间标识。

表单的标识键是由命名空间和名称以冒号(&#39;:&#39;)分隔的字符串组成的(例如：“cus:contact”)。
