# 清除事件{#purging-events}

您可以使用部署向导配置数据在数据库中存储的时间。

事件清除由工作流自动执 **[!UICONTROL Database cleanup]** 行。 此工作流清除在执行实例上接收和存储的事件以及在控制实例上归档的事件。

根据需要使用箭头更改清除设置。
