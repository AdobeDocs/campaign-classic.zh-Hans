# 数据缓冲区{#data-buffer-zone}

您可以通过取消同步选件命题计算来配置数据缓冲区以提高入站交互性能。 此配置将在实例自己的配置文件(config-Instance.xml)中执行。 如需详细信息，请参阅[此部分](../../installation/using/interaction---data-buffer.md)。

>[!NOTE]
>
>如果您使用分布式架构的交互，则此参数是必需的。
