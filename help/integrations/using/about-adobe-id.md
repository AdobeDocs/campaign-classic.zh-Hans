# 关于Adobe ID{#about-adobe-id}

Campaign用户可以使用其Adobe ID连接到Adobe Campaign控制台。 该集成具有以下优势︰

* 所有 Experience Cloud 解决方案都可以使用相同的 ID。
* 使用具有不同集成的 Adobe Campaign 时，可以记住该连接。
* 密码管理策略更安全。
* 使用联合 ID 帐户（外部 ID 提供商）。

有关不同类型的Adobe ID的详细信息，请查阅： [https://helpx.adobe.com/en/enterprise/help/identity.html](https://helpx.adobe.com/en/enterprise/help/identity.html)

>[!CAUTION]
>
>IMS实施严格由Adobe技术管理员负责。 IMS的实现是一个复杂的过程，需要一些时间。
