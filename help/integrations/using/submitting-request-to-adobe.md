# 向Adobe提交请求{#submitting-request-to-adobe}

>[!NOTE]
>
>作为v7用户，为了能够继续下一步操作，请确保在您的系统上启用了AdobeIdentity Management System (IMS) 。 [了解详情](../../integrations/using/about-adobe-id.md)。
