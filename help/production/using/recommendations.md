# 建议{#recommendations}

Adobe Campaign是一个高度事务性系统（OLTP数据库）。 这意味着基础数据库将经常更新，导致性能随时间推移而下降。 为避免此类问题，必须定期维护数据库。

>[!CAUTION]
>
>只有定期维护数据库，才能实现最佳性能。 某些RDBMS提供的自动维护不够充分，并不取代任何关系型数据库事务管理系统的深入维护。
>  
>本文中描述的步骤是建议。 维护计划是数据库管理员的责任，在遇到问题时，管理员必须是您的第一个联系人。
