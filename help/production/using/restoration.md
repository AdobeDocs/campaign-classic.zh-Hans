# 恢复{#restoration}



在干净的服务器上，恢复过程如下：

* 在已安装并配置的操作系统（网络）上，
* 安装第三方应用程序：Web服务器、JDK（如有必要）、
* 安装与源系统具有相同内部版本的Adobe Campaign二进制文件，
* 复制配置文件、跟踪日志和重定向文件，
* 创建和重建数据库，
* 启动Adobe Campaign。

有关详细信息，请参阅&#x200B;**安装指南**。
