# 恢复{#restoration}

在干净的服务器上，恢复过程如下：

* 安装和配置的操作系统（网络）上，
* 安装第三方应用程序：Web服务器、JDK（如有必要）、
* 使用与源系统相同的版本安装Adobe Campaign二进制文件，
* 复制配置文件、跟踪日志和重定向文件，
* 创建和重建数据库，
* 启动Adobe Campaign。

有关详细信息，请参阅《安 **装指南》**。
