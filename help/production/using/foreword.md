# 前言{#foreword}

此文档详细说明了Adobe Campaign的生产过程：配置、监控、升级过程、数据处理、数据库维护过程和建议。 此文档末尾提供了疑难解答部分。

生产程序必须谨慎进行，其影响必须事先充分考虑，程序必须严格执行。 它只能由专家用户执行。 我们强烈建议在开始任何生产过程之前先与Adobe Campaign联系，特别是启动可升级的步骤。
