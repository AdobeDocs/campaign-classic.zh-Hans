# 警报{#alert}

**警报**&#x200B;活动向操作员组发送消息。 它的操作方式与批准活动相同，但在这种情况下不会做出响应。

![](assets/edit_alerte.png)

警报不是永久的，因此在控制台中不可见。 要接收通知，已分配组的操作员必须具有完整的电子邮件地址。 此活动的配置与&#x200B;**Approval**&#x200B;的配置类似。 用于提醒操作符的默认投放模板是“alertAssignee”。
