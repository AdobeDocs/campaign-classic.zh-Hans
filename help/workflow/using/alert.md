# 警报{#alert}

警报 **** 活动向一组操作符发送消息。 它的操作方式与批准活动相同，但此情况下不会做出任何响应。

![](assets/edit_alerte.png)

警报不是永久的，因此在控制台中不可见。 要接收通知，已分配组的操作员必须拥有完整的电子邮件地址。 此活动的配置与“批准”的配置类 **似**。 用于提醒操作符的默认交付模板是“alertAssignee”。
