# 警报{#alert}



**警报**&#x200B;活动向操作员组发送消息。 它的操作方式与批准活动相同，但预计在此情况下不会有任何响应。

![](assets/edit_alerte.png)

警报不是永久性的，因此从控制台中不可见。 分配的组的操作员必须具有完整的电子邮件地址才能接收通知。 此活动的配置类似于&#x200B;**审批**&#x200B;的配置。 用于提醒操作员的默认投放模板为“alertAssignee”。
