# 时间约束{#time-constraint}

“时 **间限制** ”活动允许您推迟执行或放弃任务。

输入活动的标签，并指定必须暂停工作流任务的时间范围。

选中该 **[!UICONTROL Try again later if outside of execution period]** 选项后，您可以在执行时间范围之外重新启动任务。 如果希望在暂停后将永久放弃工作流操作，请取消选择此选项。
