# 隐私数据保护规定{#general-data-protection-regulation-gdpr}

默认情况下，下面详述的工作流与&#x200B;**隐私数据保护条例**&#x200B;模块一起安装。 有关此模块的更多信息，请参阅此[文章](https://helpx.adobe.com/cn/campaign/kb/acc-privacy.html)。
